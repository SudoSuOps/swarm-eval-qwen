# Datasets

**No raw data in this repo.**

Raw training/eval JSONL files live in NAS (`/data2/`) or R2 (`sb-judge/`).

This directory contains only:
- `manifests/` — dataset metadata, file lists, SHA256 hashes, pair counts
- `fingerprints/` — SHA256 checksum files for quick verification
- `splits/` — train/eval split lists (fingerprint IDs only, not data)
